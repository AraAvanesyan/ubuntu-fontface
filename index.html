Ubuntu Font Family Bower-ready

How to use it

  1. Install it from Bower

      $ bower install [--save] ubuntu-fontface

  2. Import it on SCSS/LESS/CSS source code. Assuming $BOWER_PATH=bower_components; $STYLE options are -base, -condensed and -mono; $SUFFIX options are css, less, scss (depending if you're using pure CSS, SCSS, or LESS.

      @import "$BOWER_PATH/ubuntu-fontface/ubuntu$STYLE.$SUFFIX";

  3. If you need any help (or have found any bug 🐞), please post it on /issues. Thank you!
